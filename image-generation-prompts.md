# Image Generation Prompts for Party Roles

You can use these prompts in your image generator to create consistent, "Slay the Spire" style portrait art for the agent classes.

## General Art Style Directives

_(If your generator supports negative prompts, you can add: 3d render, photorealistic, photography, text, words)_

> Slay the Spire style 2D digital art, dark fantasy aesthetic, stylized comic book illustration, thick outlines, moody lighting, card game portrait art, centered character composition, plain dark background.

---

## Where to save the images

Once you have generated the images, save them (preferably as `.png` or `.webp`) in this directory in your repository:

`public/images/roles/`

_Note: Make sure to name them in lowercase to match the class names exactly (e.g., `ranger.png`, `wizard.png`, `warrior.png`)._
